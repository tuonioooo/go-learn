## 什么是伪随机数生成?
1. 数字看起来是随机生成的,但实际上并不是 *正确答案*
2. 根据物理定律生成随机数
3. 生成伪偶数和奇数

> **1:** 计算机是确定性机器。它们无法生成真正的随机数(不像实际的物理过程)。


## 什么是种子数?
1. 两台计算机之间交换随机数
2. 用于获取 0 到种子数之间的随机数
3. 用于初始化伪随机数生成器 *正确答案*


## Go 中使用哪个包来生成伪随机数?
1. pseudorand
2. rand *正确答案*
3. random
4. randomizer


## [0, 5) 是什么意思?
1. 0 到 5 之间的数字范围(不包括 5) *正确答案*
2. 0 到 5 之间的数字范围(包括 5)
3. 只有 0 和 5
4. 只有 0 和 4

> **1:** 正确。方括号表示:"包含"。圆括号表示:"排除"。所以,[0, 5) 表示:0, 1, 2, 3, 4。这被称为"数学区间记号"。


## 为什么这个函数调用不起作用?
```go
rand.Intn(0)
```
1. 首先你应该为它设置种子
2. 它需要两个参数
3. Intn 在 [0, 0) 范围内工作。所以,同时包含 0 又不包含 0 是没有意义的。 *正确答案*

> **1:** 这不是这个错误的原因。你并不总是需要为它设置种子。
> **2:** 不,它不需要两个参数。


## 这个程序会打印什么?
注意,下面每个种子数返回的伪随机数如下:

```
种子: 0
  3 3 6 8 4 1 9 3 6 6

种子: 1
  1 1 9 3 2 4 7 6 6 6

种子: 2
  10 1 2 2 0 6 4 1 0 5
```

程序如下:

```go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	for i := 0; i < 3; i++ {
		rand.Seed(int64(i))
		fmt.Print(rand.Intn(11), " ")
		fmt.Print(rand.Intn(11), " ")
	}
}
```
1. 3 1 10 3 1 1
2. 3 6 1 6 10 5
3. 1 10 1 1 3 3
4.  3 3 1 1 10 1 *正确答案*

> **4:** 数字是根据种子数确定的。所以,这个循环分别用 0、1 和 2 为伪随机生成器设置种子。
> 
> 并且,在每次设置种子后,它调用 Intn 两次来生成两个随机数。
> 
> 所以,如果你看结果,3 3 是种子 0 的前两个数字。1 1 对应种子 1。10 1 对应种子 2。


## 如果你希望伪随机生成器在每次运行程序时都产生随机数,你应该怎么做?
1. 你需要这样设置种子: rand.Seed(rand.Random)
2. 你需要这样设置种子: rand.Seed(time.Now().UnixNano()) *正确答案*
3. 你需要这样设置种子: rand.Seed(time.Now())
