# 垃圾信息屏蔽挑战技巧

## 规则：

* 不应使用标准库函数
* 应直接操作字节来解决问题
* 使用索引、切片、追加等方式操作字符串的字节
* 保证效率：不要使用字符串连接（+运算符）
    * 应基于给定的字符串参数创建新的字节切片作为缓冲区
    * 然后在程序运行期间操作该缓冲区
    * 最后打印该缓冲区

* 仅屏蔽以 `http://` 开头的链接
* 不检查字母大小写
    * 重点是学习如何操作字符串中的字节，而非创建完美的屏蔽器
    * 例如：目前允许垃圾信息发送者通过以下方式绕过屏蔽（现阶段可以接受）：
      ```
      "这是我的垃圾页面：hTTp://youth-elixir.com"
                               ^^
      ```
* 但必须成功处理以下情况：
  ```
  输入：
  这是我的垃圾页面：http://hehefouls.netHAHAHA 再见。

  输出：
  这是我的垃圾页面：http://******************* 再见。
  ```

## 步骤：

1.  检查是否存在命令行参数。若无，则输出提示信息并退出程序
2.  创建一个与参数字符串等大的字节缓冲区
3.  循环检测 `http://` 模式
4.  将输入内容逐字符复制到缓冲区
5.  若检测到 `http://` 模式，先复制 `http://`，然后对原链接的后续部分（直至遇到空白字符）用 `*` 号替换
    例如：
    ```
    输入：
    链接：http://www.mylink.com 点击！

    输出：
    链接：http://************** 点击！
    ```
6.  将缓冲区以字符串形式打印输出
