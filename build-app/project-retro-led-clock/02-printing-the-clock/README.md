
# **目标 2：打印时钟（Printing the Clock）**

## **说明（Notes）**

* `main.go` 文件中包含了上一步的解决方案。
* `solution` 文件夹中包含本步骤的完整解决方案参考。

---

## **挑战步骤（Challenge Steps）**

1. **获取当前时间（Get the current time）**

2. **从当前时间中获取小时、分钟和秒（Get the current hour, minute and second from the current time）**

3. **使用 digits 数组，从中取得对应的数字，创建 clock 数组（Create the clock array by getting the digits from the digits array）**

4. **使用 clock 数组打印出时钟（Print the clock by using the clock array）**

5. **创建一个分隔符数组（separator array），它也是一种占位符类型（Create a separator array — it’s also a placeholder type）**

6. **将分隔符加入到 clock 数组中的正确位置（Add the separators into the correct positions of the clock array）**
