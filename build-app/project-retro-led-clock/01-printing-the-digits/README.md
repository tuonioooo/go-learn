# 目标 1：打印数字（GOAL 1: Printing the Digits）

## 挑战步骤（Challenge Steps）

1. **定义一个新的占位符类型（placeholder type）**

2. **创建从 “zero” 到 “nine” 的所有数字图形**

   你可以使用以下字符来构建数字时钟样式：

   ```
   数字字符（Digit character）     ：█
   分隔字符（Separator character）：░
   ```

3. **把它们放入 `digits` 数组中**

4. **并排打印这些数字**

   实现方式：

   1. 循环遍历每个数字的所有行（lines）
   2. 每一行中，按顺序打印所有数字对应的该行内容
   3. 别忘了在每个数字后面加一个空格
   4. 别忘了在每行末尾打印换行符

---

### 示例（EXAMPLE）

假设你要打印数字 **10**，效果如下：

```
      ██   ███   <--- 在打印完所有数字的“单行”后，要换行
       █   █ █
       █   █ █
       █   █ █
      ███  ███
         ^^
         ||
         ++----> 数字之间要加空格
```
